# Calendar App
A multi-platform calendar app to help you schedule and balance your life. Using a simple UI focused on simplicity and efficiency.

## Features
- [x] Multi-platform account and schedule view
- [x] Share static or dynamic calendar for other users or guests
- [x] Home view: A special view method builty to mantain the calendar open on a smart screen. [(Insipiration)](https://youtu.be/RWjvJq4Zabk)
- [x] Weather based on preselected user location or current user location.
    (e.g. Options specific for user taste such as winter sports with infor regarding snow, or aviation geeks with aviation weather information)
- [x] Statistics regarding user time usage and possible ways to improve it
- [x] Group schedules (family, friends, etc.)
    (e.g. Allowing multiple people to add, remove and edit apointments on shared calendar)
- [x] Business appointments (commerce, office meetings, etc.)
    (e.g. Businessess would have a configurable interface and settings to allow different people to make reservations or appointments such as barber shops, restaurants, offices for reserving meeting rooms)
- [x] Business Map
    Businessess which have appointments open to public would be able to draw the map of their business space to allow users to visually select what location or space they want to reserve.
    (e.g. An office building which has multiple meeting rooms could have a map of each room for easy visualization of space usage)

## MVP
The objective of the minimum viable product is to design a viable and expandable UI and database firstly.

- Create UI
    - Create web UI for desktop and mobile
    - Desktop App
- Database schema based on features expected for first year of operation  
    - Include all basic personal user features
    - Base business interaction for personal user schema
    - Export/import functions for database and case of needed updates
    - Search for viable methods to use a database tech, that if needed, when updated would not erase existing data
